## Makefile

Génèrer les tests unitaires pour le lot A et exécuter `valgrind` dessus

```
make vg_a
```

Générer les tests unitaires pour le lot A

```
make u_a
```

Générer la documentation Doxygen du projet

```
make Doxygen
```

Nettoyer le projet

```
make Clean
```
